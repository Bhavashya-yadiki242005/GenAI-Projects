# GenAI-Projects

This repository contains a curated collection of end-to-end projects in **Machine Learning, Natural Language Processing, Semantic Search, and Agentic AI systems**. Each project is designed to demonstrate practical problem-solving skills, clear implementation, and real-world applicability using modern AI frameworks and tools.

The projects range from classical NLP techniques to advanced **Agentic RAG systems**, showcasing both foundational understanding and cutting-edge Generative AI workflows.

---

## Projects Overview

---

## News Article Categorizer (NLP Text Classification)

An automated system to classify news articles into predefined categories such as **business, politics, sports, entertainment, and technology** using traditional NLP and machine learning techniques.

### Key Highlights
- Text preprocessing using tokenization and stopword removal
- Feature extraction using Bag of Words (BoW) and TF-IDF
- Classification using Multinomial Naive Bayes
- Model performance comparison between BoW and TF-IDF
- Custom input prediction support

### Technologies Used
- Python, Pandas, NumPy
- NLTK
- Scikit-learn

### Use Cases
- Media monitoring
- Content recommendation systems
- Topic-based news filtering
- Preprocessing pipeline for sentiment analysis systems

---

## Semantic Document Search using FAISS

A high-performance semantic search engine that retrieves relevant document content based on **meaning rather than exact keyword matching**.

### Key Highlights
- Supports PDF, DOCX, and TXT documents
- Text chunking for improved retrieval accuracy
- Sentence embeddings using transformer models
- FAISS indexing with cosine similarity
- Query-based semantic document retrieval

### Technologies Used
- Python
- FAISS
- Sentence Transformers
- PyMuPDF, python-docx

### Use Cases
- Knowledge base search
- Enterprise document retrieval
- Research paper and policy document search
- AI-powered internal search engines

---

## Math Problem Solver using LangChain Agents and OCR

A web application that solves mathematical problems directly from images using **OCR and LLM-based reasoning**.

### Key Highlights
- Image-to-text extraction using OCR
- Step-by-step reasoning using LLMs
- Automated problem understanding and solution generation
- Interactive web interface using Gradio

### Technologies Used
- LangChain
- OpenAI / Gemini LLMs
- Pytesseract (OCR)
- Gradio
- Python Imaging Library (PIL)

### Use Cases
- Educational tools
- Automated homework assistance
- Assistive technology for handwritten problem solving

---

## Agentic RAG System using LlamaIndex

An advanced **Agentic Retrieval-Augmented Generation (RAG)** system where an autonomous agent dynamically decides how to retrieve and generate responses using multiple tools.

### Key Highlights
- Agent-driven decision making
- Document retrieval using LlamaIndex
- Multi-tool support (Document Search, Calculator, Wikipedia)
- Conversational memory for follow-up queries
- Modular and scalable architecture

### Technologies Used
- LlamaIndex
- LangChain
- OpenAI GPT models
- Wikipedia API
- Python

### Use Cases
- AI knowledge assistants
- Enterprise QA systems
- Intelligent chatbots with reasoning capabilities
- Research and decision support systems
